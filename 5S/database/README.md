# Base De Datos 5S

Las tablas 5S estan dentro del esquema Supabase multiempresa principal:

- `configuracion_5s`
- `catalogos_5s`
- `bodegas_5s`
- `responsables_5s`
- `cronograma_5s`
- `checklist_items_5s`
- `inspecciones_5s`
- `inspeccion_items_5s`
- `evidencias_5s`

Todas deben tener `empresa_id` y consultarse filtrando por empresa.
